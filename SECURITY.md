## Reporting a Vulnerability

We take the security of our project seriously. If you discover a vulnerability, please follow these steps:

1.  **Do not** create a public GitHub issue for the vulnerability.
2.  Email the details of the vulnerability to `security@example.com` (replace with actual contact if available, or instruct to use private disclosure).
3.  Include a description of the issue, steps to reproduce, and the potential impact.

We will acknowledge your report within 48 hours and work to provide a fix as soon as possible.
